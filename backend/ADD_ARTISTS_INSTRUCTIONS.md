# Instructions to Add More Artists

## Quick Steps:

1. **Stop the backend server** in your Anaconda Prompt (Press Ctrl+C)

2. **Run the script**:
   ```bash
   python add_more_artists.py
   ```

3. **Restart the server**:
   ```bash
   uvicorn main:app --reload
   ```

4. **Refresh your browser** at http://localhost:5174/artists
